# SiteFlow — Diseño Web que Convierte

Landing page profesional desarrollada para una marca de diseño web enfocada en generar clientes, mejorar la percepción de negocio y aumentar conversiones.

## 🚀 Enfoque

SiteFlow no se centra solo en diseño visual.

El objetivo es crear páginas web que:
- generen confianza
- comuniquen profesionalismo
- diferencien del competidor
- conviertan visitas en clientes

## 🧠 Concepto del proyecto

La web está diseñada bajo una idea clara:

> El diseño no es estética. Es estrategia.

Cada sección, color y estructura está pensada para impactar en la decisión del usuario.

## 🎯 Características principales

- Diseño moderno y minimalista
- Estructura orientada a conversión
- Hero con propuesta de valor clara
- Sección de estilos visuales (educación del cliente)
- Proceso de trabajo profesional
- FAQ estratégicas
- CTA fuerte hacia WhatsApp
- Diseño responsive (mobile-first)
- SEO on-page optimizado

## 🎨 Diseño y experiencia

- Paleta oscura premium
- Tipografía moderna (Inter)
- Uso de espacios y jerarquía visual
- Animaciones suaves con GSAP
- Interacciones limpias y profesionales

## 🛠️ Tecnologías utilizadas

- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- GSAP (animaciones)
- Lucide Icons
